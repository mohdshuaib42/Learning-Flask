Learning Flask 🚀
A hands-on repository to learn Flask, the lightweight Python web framework. This project walks through building a simple web app step-by-step, from basics to deployment.

📖 About This Project
This repo documents my journey learning Flask. It includes:

Basic "Hello World" app

Routing and templates

Forms and database integration (SQLite)

Authentication basics

Deployment to Render/Heroku

Perfect for Flask beginners!

🛠️ Tech Stack
Backend: Flask 3.x, Python 3.10+

Database: SQLite (with Flask-SQLAlchemy)

Frontend: Jinja2 templates, Bootstrap 5

Testing: pytest

Deployment: Render (free tier)
